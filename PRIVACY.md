Privacy Policy for YouTube Music Liked Toast Dismiss

Data Collection
This extension does NOT collect, store, or transmit any user data.

Permissions
The extension only requires access to https://music.youtube.com/* to run the content script that auto-dismisses the toast. No other permissions are requested.

Third-Party Services
This extension does not use any third-party services, analytics, or tracking.

Contact
For questions or concerns about privacy, open an issue on the GitHub repository.
